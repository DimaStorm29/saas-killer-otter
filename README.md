# Otter Killer - Free Transcription Script

Stop paying $120/year for Otter.ai. This script gives you unlimited, private transcription for free.

## What It Does

- Transcribes audio files using OpenAI's Whisper model
- Saves output to text files locally
- No cloud upload - your data stays on your machine
- Works with any audio format supported by Whisper

## Requirements

```bash
pip install openai-whisper
```

## Usage

1. Save your audio file as `meeting_notes.mp3` in the same folder as this script
2. Run the script:

```bash
python otter_killer.py
```

3. Your transcription will be saved as `meeting_notes.txt`
